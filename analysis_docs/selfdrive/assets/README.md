# selfdrive/assets 概要

このディレクトリはopenpilotのUIや音声、フォント、画像など「リソース（アセット）」を一元管理する場所です。主に以下の役割を持ちます。

## 目的・背景
- openpilotの画面表示や音声案内、各種UI部品の見た目・音・多言語対応などを支える素材をまとめて管理
- Qt（C++/Python）アプリケーションで利用するリソースファイル（.qrc）や、画像・音声・フォント・SVGなどの実体ファイルを格納
- ビルド時にリソースとしてバイナリに埋め込まれるものも多い

## 主な構成
- `assets.qrc`, `translations_assets.qrc` : Qtリソース定義ファイル。どの素材をアプリに埋め込むか記述
- `assets.cc` : C++でリソースを扱うための実装
- `compress-images.sh`, `prep-svg.sh` : 画像やSVGを最適化・変換する補助スクリプト
- `fonts/` : UIで使うフォントファイル
- `icons/` : ボタンやUI部品のアイコン画像
- `images/` : UIや地図などで使う汎用画像
- `sounds/` : 操作音や警告音などの音声ファイル
- `body/`, `offroad/`, `training/` : 特定画面や機能ごとの素材

## 使い方の例
- UIの見た目を変えたい場合は`icons/`や`images/`の画像を差し替える
- 新しい言語対応を追加したい場合は`translations_assets.qrc`を編集
- 画像を追加したら`compress-images.sh`で最適化

## 注意点
- 画像や音声はライセンスに注意して追加・変更すること
- Qtリソース（.qrc）はビルド時に自動でバイナリに変換される
